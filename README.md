## Descripción

Este sitio web fue creado con el objetivo de representar mi proyecto final de la media técnica en programación de software durante el año 2025. El proyecto fue denominado como **el mejor proyecto de todas las medias técnicas de los estudiantes hasta ese año**.

El sitio web está enfocado en la **venta de perfumes** y está conectado a una base de datos en XAMPP con 11 tablas (no se muestran todas ya que algunas fueron descartadas durante el desarrollo):

- **compras** – 7 registros  
- **devoluciones** – 1 registro  
- **inventario** – 2 registros  
- **marcas** – 3 registros  
- **menu** – 7 registros  
- **perfumes** – 3 registros  
- **resenas** – 81 registros  
- **solicitudes_admin** – 0 registros  
- **usuarios** – 8 registros  
- **usuariosss** – 0 registros  
- **ventas** – 3 registros  

> Nota: La base de datos es obligatoria para que el sitio web funcione correctamente. Algunas tablas fueron descartadas durante el desarrollo.

---

## Uso

1. Configura XAMPP y crea la base de datos proporcionada.  
2. Asegúrate de que todos los archivos del proyecto estén en la carpeta del servidor (por ejemplo, `htdocs`).  
3. Abre el sitio web en tu navegador.  

> Puedes hacer pequeñas modificaciones si conoces el código y su funcionamiento.
